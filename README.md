# 📦 Установка модов

## 🟢 Вариант для обычных пользователей (без консоли)

### Установка:

1. Скачать архив:  
   [https://github.com/Mag1cMouse/tech-modpack/archive/refs/heads/main.zip](https://github.com/Mag1cMouse/tech-modpack/archive/refs/heads/main.zip)
2. Разархивировать содержимое в папку `mods`  
   _(предварительно очистив её)_

### Обновление:

1. Запустить файл `git-pull.bat` (двойной клик)
2. Готово — моды обновятся 🎉

## 🧑‍💻 Вариант через Git (консоль)

### Установка:

1. Скачать Git:  
   [https://github.com/git-for-windows/git/releases/download/v2.53.0.windows.2/Git-2.53.0.2-64-bit.exe](https://github.com/git-for-windows/git/releases/download/v2.53.0.windows.2/Git-2.53.0.2-64-bit.exe)
2. Удалить всё из папки `mods`
3. Открыть консоль в папке `mods`
4. Выполнить команду:

```
git clone https://github.com/Mag1cMouse/tech-modpack.git .
```

5. Готово — можно играть 😎

### Обновление:

В папке `mods` выполнить:

```
git pull origin main
```
